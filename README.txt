ALMA — Capstone web app (UZP 02.02 · Vicálvaro, Madrid)
========================================================
4 pages, one linked model:
  • index.html            – Dashboard (start here)
  • land-value.html       – Residual land value (developer)
  • land-distribution.html– How the land / GFA splits
  • city.html             – Municipality 30-year P&L

HOW TO USE
----------
A) Just open it:      double-click index.html
B) Full interactivity (live sync between pages):
   in this folder run:  python3 -m http.server 8000
   then open:           http://localhost:8000
C) Publish online:    drag this folder onto https://app.netlify.com/drop
                       (or any static host / GitHub Pages) → you get a share URL.

Needs an internet connection for the Inter font (Google Fonts); everything else is self-contained.
All figures are a working capstone model, not appraised values.
